# xml_splitter

## Описание
`Exel_creation.py` — это Python-скрипт поиска нужных ключевых слов в файлах. Скрипт поддерживает обратную логику и поиск по количеству вхождений. Результаты сохраняются в Excel-файл для дальнейшей работы с тестовой системой Vanessa.
`XML_Lodader.py` - для выгрузки конфигурации 1С в XML 

## Возможности
- Автоматическая обработка XML-файлов конфигурации 1С
- Поиск ключевых слов с учетом количества вхождений
- Обратная логика поиска (инвертированный результат)
- Генерация отчета в Excel
- Поддержка различных типов объектов: модули форм, модули объектов, регистры сведений, регистры накопления и т.д.
- Вывод цветных сообщений в терминал с помощью `colorama`
